# justforfun

This is a naive exploration into the evolution of orphan GPCRs.

A natural next step for this project would be to analyze orphan GPCRs across species: for each genome, plot the number of orphan GPCRs (and optionally normalize by genome size) across evolutionary time, comparing trends across diverse organisms. This could highlight bursts of GPCR innovation, lineage-specific expansions, and relationships between genome size and the prevalence of orphan receptors.
